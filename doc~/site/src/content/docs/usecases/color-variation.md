服にカラバリprefabがあるとき、この方法でさくっとカラバリメニューが作れます。

### 1. 入れたいカラーバリエーションのprefabをアバターにいれる

![](../../../assets/imgs/colorvariation-1.png)

### 2. デフォルト色のprefabのメッシュを選択し、選択肢メニューでマテリアル変更にチェックを入れる

![](../../../assets/imgs/colorvariation-2.png)

`Materials`の「+」から各メッシュの項目を追加

![](../../../assets/imgs/colorvariation-3.png)

### 3. 2番目選択肢の色のprefab名をデフォルト色のprefab名にする

デフォルト色prefab名を一旦変えて……

![](../../../assets/imgs/colorvariation-4.png)

カラバリのprefab名をデフォルト色のものにする

![](../../../assets/imgs/colorvariation-5.png)

### 4. 選択肢の抽出ボタンを押す

抽出ボタンを押すと当該選択肢の項目に値がコピーされます

![](../../../assets/imgs/colorvariation-6.png)

### 5. この手順を他の選択肢でも行う

「名前を揃えて抽出ボタンを押す」を繰り返してゆきます。

![](../../../assets/imgs/colorvariation-7.png)

### 6. 最後にデフォルト色prefabの名前を戻してカラバリprefabを削除する

![](../../../assets/imgs/colorvariation-8.png)
